## Serve Directory

~~~shell
cd folder/you/want/to/host/
nousb
// Custom port (default is 8080)
nousb -p=80
~~~

## Serve Directory With Custom Port

~~~shell
cd folder/you/want/to/host/
nousb -p=80
~~~

## Using

Enter the IP:Port on the other device's browser to load up the web ui.
